## 生活记录

### 01 

![image-20230213002637340](https://s2.loli.net/2023/02/13/dVbZJeswCqcFguR.png)

自己长时间缺失了像样一点的生活记录了，虽然有坚持在写日记，但大部分内容其实都是任务安排、情感垃圾。甚至连像样一点的情绪日记也写的不是特别好。

当然，有在记其实也挺不错的了，最糟糕的那段时间，往往是连自己在想什么都察觉不到的时间。

<br/>



突然又想回忆一下上一个暑假到大概大二上前两个月的阶段，那个时候用公众号来写记录。

现在抽离出来看，确实觉得那时候写的记录挺憋屈的，好像总会有那么一点写给别人看的感觉。

写在自己博客里面感觉就不一样了，一时半会我也说不出不一样在哪。好像就是更私密，更随便，最重要的是，**更真实**。

<br/>





突然有了现在这段文字，是因为前面几天有关博客那篇记录里面说到过，突然又找回了一点记录的感觉。

我在看别人博客中的生活记录的时候，我没什么特别的想法，就是有种心灵感应般的“我本也应该是这样”。

也有在那之中没说到的地方，就是真的，有时候半夜确实就是，怎么形容，就是好像周围的黑暗要把你吞噬掉，有时候还下着大雨，就好像周围所有的一切都和自己的内心连成一片了。

没写记录的时候，正常的情况可能习惯于看点有些意味的番，好像这样就能填满我的内心。

但是也有其他的情况，可能会因为家里的变故控制不好自己情绪而痛哭，可能会因为控制不好压力而选择逃避找一些刺激。这样就睡不着觉了。但也总会趋于平静的，想来不爱睡觉的朋友还挺多的，有时候深夜聊聊以前的事情还挺快乐的。![image-20230213001446541](https://f.pz.al/pzal/2023/02/12/c66b2361e8f6a.png)



写写像这样的记录就挺好的，好像一切的一切都有了归宿。

从来不是为了追求什么而写的，只是本来就会有很多这样的想法溢出来。

写记录是我最容易进入心流的方式。

不管是回忆，还是对最近生活的回味，抑或是对一些事情的看法。

我都很乐意把它们记下来。这就是我最想要的记录的感觉。



### 02 我一直以来都很幸运

（可以看到我上面的 Verse 就没加标题，这里的 Verse 就没加图片，就是想验证一下“这里都是我完全随心所欲想写的”）

<br/>

然后就是特别想回味一下寒假在做的事情，要是不做点记录，真的不知不觉就过去了，就好像很容易在这之中丢掉很多东西。



整个寒假的时间基本就是花在做[青训营](/blog/tags/青训营)的后端项目了，虽然时间利用率还是不是很理想，我还是有很多时间都浪费掉了，但是总的来说，这真的是我感觉最有收获的一次了。

一个一直有的想法是，不管能不能成功，给自己在假期找点东西支撑着，想要有收获就得稳定。

**找到自己真正想做的事情，并且找一个稳定的环境就好了。**

<br/>



然后大概在这几天我又开始有点懈怠了，好像稍微有点厌倦了。

这个时候想起来之前第二次字节青年的志愿者前辈跟我说的：

“要是发现自己其实并不感兴趣了，不妨想想自己是为什么开始的；要是本来就是不认同的东西，那就没有坚持下去的必要。”

**去发现自己的合理性**

<br/>



说着说着又偏题了，不过这才是记录嘛，标题就当个部分提炼好了。

主要是想说，这次项目遇到的队友实在太棒了。

bls说我积德终于有回报了，想来也是，有的时候不要太不敢面对好像本不该属于自己的事情。

接受并且好好对待也是需要勇气的。



### 03 结项总结与反思



这次队长当的是真的很有水分的，继续感谢足够靠谱的队友能让我正常走了下来。



这里主要写写个人向的一些反思，其余基本都包括在这个里面了：[结业答辩文档](https://sygubn5jgz.feishu.cn/docx/QJYNdnc3soxIPJxDiqLc3iDXn2e)

（要是有路过的大佬感兴趣欢迎指出问题）

<br/>



#Keep

- 值得肯定的是我至始至终没有考虑过逃避，就算情况差到极致了，我也考虑的是平稳接受下来。
- 任务管理上做得尚可。
  - 有新想法往后推，先做好当前的事情。
  - 遇到困难就好好拆解成一系列小目标。

- 队友之间的相互支持是很重要的
  - 作为支柱需要给大家稳定的反馈
  - 学会求助也是一种能力


#Problem

- 虽然最开始的时候做的项目规划，做的文档整理都还不错，但是后期居然**就没怎么回顾过**了，这是队长最应该做的事情。

  这些地方就不要太依靠别人能帮你记住，这就导致整个队伍长时间缺失了一些比较重要的官方信息。

  就算只有那么几个文档，想把信息全部利用起来还是不容易的，所以回顾就很重要。

- 总想着队长一定要做最多的事情，把所有的事情都承担下来。

  这是个老问题了。

  其实到最后队友承担的我自己感觉是比我多很多了，这样也挺好的，有时候就接受现在合理的情况，至少队友把“从整体上把握好项目”这个重任交给我了，那我只要做好该做的事情就行了。

- 不够自信

  不过自信真没法当队长。最早的时候可能就是一种一个分支都不敢合并，问题都不敢问的感觉。那显然完全没必要当队长的。坚信自己是对的，没有必要考虑太多。

- 没有做比较具体的项目规划。

  上面说的做得还不错其实有点放屁，还得是队友们都比较善良。不过当时的自我感觉良好从也是有一部分好处的，让我重新收获了一些前进下去所必要的一些信心。

  现实情况是项目的时间过了一半多才开始写出来稍微像样一点的文档，导致时间安排已经是比较紧的了。

  ![image-20230213165623913](https://f.pz.al/pzal/2023/02/13/fd4773c06ee4e.png)

- 开发架构的思路不够清晰，自己都想不到哪个函数应该在哪个具体的层。发现了UML确实是很有用的一个东西，值得学习和研究

- 时间一拉长就比较任务化了，不够有动力，比较痛苦。回归原点的话应该是自己本身想尝试这件事情，而不是为了交一个自己不是很愿意做的项目。

#Try

- 多做整体上的规划

  - 从一开始就把哪些是自己该做的，哪些是不该做的规定好。
    就比如说，每个人的代码肯定是都得懂大概是干嘛的，但具体的实现没商量好最好就不要插手。这二者还有一种微妙的平衡
  - 更重要的是，就算自己清楚“大部分情况下计划赶不上变化，而且不合理的计划容易让自己摆烂”，但一份大致的计划依然是有必要的。

- 优化日常的工作流程。

  看起来每天都累成狗，实际上时间利用效率真的不高。偶尔也有效率高的时候。**分解任务，定期休息**至关重要。要是天天想着打无尽模式大概率就开摆了。

- 做好压力控制。

  虽然队长有责任，但是也要适可而止。
  
- 队长承担的重任应该是把大任务最合理地分解到每个人，使得最终目的完成的效率最高

  

### 05 说说我寒假的收获

这个寒假带给我的变化是很多的

- 完整的项目经历和体验，做成一件事情的能力。

  这是我自己接触一个比较完整的项目，虽然量化收获可能不多，但我更原因坚信，**当我有了做完一件事的能力以后，我也能获得挑战所有事情的能力。**

- 日常生活逐渐充实，找到了自己很多新的兴趣，每天都能做一些有趣的探索，基本上很难说无聊没有什么可玩的东西了。

  那种觉得每天都有无限可能的感觉实在是太棒了

- GPT 的变革，搜索3.0时代的降临，深刻享受学习成本降低的恩惠。

<br/>

还有一个小点是也开始适应吸收英语材料了，对于简单一点的英语材料在状态不算太差的时候都能吸收。这个需要继续坚持。

<br/>

我好像不再那么强迫自己了，在那次沟通以后，慢慢发现并接受了很多属于自己的合理性。一些事实上的改变是，很多时候我不在追求不适合我的学习工作方法，这使得我现在做很多事情都比较无痛，只是自己想试试而已。

<br/>

还有一个值得单独说一说的，一个非常大的收获是，认识了非常好的朋友



很多时候就是真的忍不住会感慨这种缘分，未曾谋面就是凭借互联网认识的这样一个人，最后却能够建立如此深的联系。

有的时候很多巧合就很让人惊喜，新认识的两个好兄弟中，一个是家乡离得不远的，一个是和自己读的相同专业的。



有时候居然也能感受到自己也有一点“聚拢人心的力量”![](https://picx.zhimg.com/80/v2-6ddea37aaea32069f63d19e0578691bf_1440w.webp?source=c8b7c179)

当然这还是有点自我膨胀和中二了。我主要想描述的那种感觉，还是交朋友给我带来的那种反馈吧。我好像能够清楚的感觉到，**当我愿意善待这个世界的时候，这个世界也在善待我**
